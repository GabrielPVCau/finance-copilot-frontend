Traduzimos operações em narrativa de crédito

O Copiloto Financeiro é um MVP construído em ambiente acadêmico para ajudar PMEs a contar, com dados reais, por que merecem melhores condições de crédito. Estamos convidando empresas para co-criar o piloto conosco.

  • Conecte dados financeiros com segurança e mantenha controle sobre o que será compartilhado.
  • Use um tradutor de negócios que transforma fluxo de caixa em argumentos compreensíveis para analistas humanos.
  • Receba feedback guiado e ajude a definir quais dados compõem o dossiê ideal para PMEs.

A armadilha que queremos desarmar

Para milhões de PMEs, o Score ignora sinais de vitalidade presentes no ERP, no Open Finance e até nos boletos pagos em dia. Queremos testar se uma narrativa baseada nesses dados muda a conversa com o banco.

Fluxo real ignorado

Recebíveis confirmados valem menos que um atraso de meses atrás. O piloto busca provar que o histórico recente deve pesar mais.

Negociação assimétrica

Enquanto o banco leva mais de 40 dias para responder, o empreendedor precisa de capital para ontem. Usaremos o MVP como tradutor entre as duas pontas.

Credibilidade portátil

Coletando uma vez, o empresário deveria poder apresentar o mesmo dossiê para vários agentes financeiros. A hipótese do Copiloto é validar esse reaproveitamento.

Como pretendemos construir o dossiê

Simplificamos em três etapas que queremos validar com você. Nada de jargão secreto: apenas passos que façam sentido para quem vive a operação.

1. Conexão segura de dados

Você escolhe quais fontes abrir (Open Finance, planilhas, ERP) e definimos juntos o escopo mínimo viável. Tudo ficará registrado para auditoria acadêmica.

  • Transparência sobre quais dados foram sincronizados
  • Possibilidade de desconectar a qualquer momento

2. Análise inteligível

Aplicamos regras visuais simples (tendência, sazonalidade, concentração) para contar a história do seu fluxo de caixa.

Primeira versão em planilhas vivas

3. Geração do dossiê

Comparamos os dados com critérios públicos (Serasa, Bacen) e sugerimos uma narrativa pronta para copiar e apresentar.

Testes com diferentes bancos

A mesma história precisa fazer sentido para cooperativas e fintechs. Mediremos como cada instituição interpreta o dossiê.

Próximos experimentos

Queremos testar alertas preventivos, simulações de prazo e templates por segmento (agro, varejo, serviços). As primeiras PMEs podem priorizar o backlog.

Dúvidas frequentes

Transparência radical

Quanto custa o Copiloto?

Durante a fase piloto, o acesso à plataforma e a geração do dossiê serão gratuitos para as PMEs selecionadas. Queremos validar o produto e colher feedbacks. No futuro, planejamos um modelo de assinatura acessível somado a uma taxa de sucesso apenas quando o crédito for aprovado.

É seguro conectar Open Finance?

Sim. Trabalhamos apenas com conexões de leitura e registramos cada permissão concedida. Durante o MVP você poderá simular com dados fictícios antes de autorizar qualquer integração real.

Posso usar com meu banco atual?

Nosso objetivo é que o dossiê seja aceito por diferentes instituições. Nesta etapa, coletamos feedback de bancos e cooperativas parceiros para entender como adaptar a narrativa. Você poderá levar o material gerado para seu banco atual e contar pra gente como foi.

Quem pode participar do piloto?

Buscamos PMEs com faturamento acima de R$ 150 mil/mês, histórico de vendas consistente e abertura para compartilhar dados de forma acompanhada. Se esse é o seu caso, preencha o formulário conversacional e vamos avaliar juntos.
