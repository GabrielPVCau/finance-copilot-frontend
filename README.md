# Finance Copilot Frontend

Interface moderna para o gerenciamento e acompanhamento financeiro pessoal.

## Funcionalidades
- Dashboard interativo para visualização de gastos.
- Planejamento financeiro assistido.
- Interface responsiva com design clean.

## Tecnologias
- HTML5 / CSS3
- JavaScript (Vanilla)
- Google Fonts (Outfit)

## Estrutura
O código fonte principal da interface está localizado na pasta `/docs` para facilitar o deploy via GitHub Pages.
